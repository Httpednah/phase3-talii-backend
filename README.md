# Talii Backend API

Backend for **Talii**, a local experiences booking platform. Built with **FastAPI** and **SQLAlchemy**, using a SQLite database.

---

## Features

- Manage **Categories**: Create, read, update, delete.
- Manage **Experiences**: Create, read, update, delete.
- Manage **Bookings**: Users can book experiences and view their bookings.
- Manage **Reviews**: Users can leave ratings and comments.
  

---

## Tech Stack

- Python 3.8+
- FastAPI
- SQLAlchemy
- SQLite
